Clinical Table Component

A comprehensive table component with Tabulator integration for clinical data display.

Patient List

Manage and view patient information

Showing 1 to 10 of 25 entries

Features

  • Sorting: Click column headers to sort data
  • Filtering: Use search and filter buttons
  • Pagination: Navigate through large datasets
  • Responsive: Adapts to different screen sizes
  • Actions: Row-level action buttons
  • Status Indicators: Visual status representation

Usage

<div class="batcat-table-container">
    <div class="batcat-table-header">
        <h3 class="batcat-table-title">Table Title</h3>
        <!-- Search and filters -->
    </div>
    <div id="batcat-table"></div>
    <div class="batcat-table-pagination">
        <!-- Pagination controls -->
    </div>
</div>
