Features

  • Text Inputs: Standard text, email, phone, and date inputs
  • Select Dropdowns: Custom styled select elements
  • Checkboxes: Multiple selection with custom styling
  • Radio Buttons: Single selection with custom styling
  • Textarea: Multi-line text input with resize control
  • Toggle Switch: Binary on/off control
  • Validation States: Error, success, and disabled states
  • Icons: Support for input icons

Usage

<div class="batcat-input-group">
    <label class="batcat-input-label required">Field Label</label>
    <input type="text" class="batcat-input" placeholder="Enter value">
    <div class="batcat-input-help">Help text</div>
</div>

Validation

// Add error state
