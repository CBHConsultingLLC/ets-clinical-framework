Clinical Header Component

A responsive header component designed for clinical applications with modern healthcare UI patterns.

Features

  • Responsive Design: Adapts to different screen sizes with Bootstrap 5
  • Brand Integration: Customizable logo and brand colors
  • Navigation Menu: Primary navigation with dropdown support
  • User Actions: Notifications, user profile, and settings
  • Accessibility: ARIA compliant with proper keyboard navigation

Usage

<header class="batcat-header">
    <div class="container-fluid">
        <nav class="navbar navbar-expand-lg">
            <div class="container-fluid">
                <a class="navbar-brand" href="#">
                    <i class="fas fa-hospital me-2"></i>
                    Your Brand
                </a>
                <!-- Navigation content -->
            </div>
        </nav>
    </div>
</header>

Customization

  • Modify CSS variables for color scheme
  • Add/remove navigation items
  • Customize user actions and dropdowns
  • Integrate with your authentication system
