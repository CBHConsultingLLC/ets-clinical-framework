Features

  • Image Gallery: Grid layout with hover effects and actions
  • Image Overlay: Zoom, download, and fullscreen actions
  • Video Player: Custom controls with progress bar
  • Charts: Interactive charts using Chart.js
  • Loading States: Visual feedback during content loading
  • Responsive Design: Adapts to different screen sizes
  • Accessibility: Proper alt text and ARIA labels

Usage

<div class="batcat-media-container">
    <div class="batcat-media-header">
        <h5 class="batcat-media-title">Media Title</h5>
    </div>
    <div class="batcat-media-content">
        <div class="batcat-image-container">
            <img src="image.jpg" alt="Description" class="batcat-image">
            <div class="batcat-image-overlay">
                <!-- Action buttons -->
            </div>
        </div>
    </div>
</div>

Chart Integration

const ctx = document.getElementById('chart').getContext('2d');
const chart = new Chart(ctx, {
    type: 'line',
    data: {
        labels: ['Jan', 'Feb', 'Mar'],
        datasets: [{
            label: 'Data',
            data: [12, 19, 3],
            borderColor: '#3b82f6'
        }]
    }
});
