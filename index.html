mPage Core Functions & APIs

Essential Cerner mPage functions and utilities for PowerChart development

Core mPage Functions

MPageView()

Base class for all mPage views. Provides core functionality for rendering, resizing, and managing mPage components.

function MyMPage() {
    MPageView.call(this);
}
MyMPage.prototype = new MPageView();
MyMPage.prototype.constructor = MPageView;

renderView()

Renders the mPage view content. Called automatically when the mPage is displayed.

MyMPage.prototype.renderView = function() {
    var html = '<div class="mpage-content">Content</div>';
    this.getElement().html(html);
};

resizeView()

Handles mPage resizing events. Called when the browser window or mPage container is resized.

MyMPage.prototype.resizeView = function() {
    // Handle resize logic
    this.adjustLayout();
};

CCL Integration

MP_Core.XMLCCLRequest()

Makes CCL requests to the Cerner backend. Used for data retrieval and business logic execution.

var request = new ScriptRequest();
request.setProgramName("MY_CCL_PROGRAM")
    .setParameterArray(["param1", "param2"])
    .setResponseHandler(function(response) {
        // Handle response
    })
    .performRequest();

CCLEVENT()

Handles CCL events and callbacks. Used for asynchronous operations and event-driven programming.

CCLEVENT("EVENT_NAME", function(data) {
    // Handle event data
    console.log("Event received:", data);
});

CCLLINK()

Creates links to other Cerner applications or mPages. Handles navigation and deep linking.

CCLLINK("APPLICATION_NAME", {
    param1: "value1",
    param2: "value2"
});

Patient Context

getPatientId()

Retrieves the current patient ID from the mPage context.

var patientId = getPatientId();
console.log("Current patient:", patientId);

getEncounterId()

Retrieves the current encounter ID from the mPage context.

var encounterId = getEncounterId();
console.log("Current encounter:", encounterId);

getProviderId()

Retrieves the current provider ID from the mPage context.

var providerId = getProviderId();
console.log("Current provider:", providerId);

UI Controls

MPageControls.Control

Base class for all mPage UI controls. Provides common functionality for form elements and interactive components.

var control = new MPageControls.Control();
control.setElement($("#my-element"));
control.setValue("default value");
control.render();

Selector

Dropdown/select control for mPages. Handles option management and selection events.

var selector = new Selector($("#my-select"), {
    displayField: "name",
    valueField: "id",
    options: dataArray
});
selector.init();

TemplateEngine

Template rendering engine for generating HTML content dynamically.

var template = new TemplateEngine.Template(function(data) {
    return '<div class="item">' + data.name + '</div>';
});
var html = template.render({name: "John Doe"});

Data Management

HealtheIntent.Utils.serviceRequest()

Makes HTTP service requests to external APIs. Used for integrating with HealtheIntent services.

HealtheIntent.Utils.serviceRequest("SERVICE_NAME", "endpoint", {
    personId: getPatientId(),
    providerId: getProviderId()
}).then(function(data) {
    // Handle response data
});

serializeFormToJson()

Serializes form data to JSON format for easy data handling.

var formData = HealtheIntent.Utils.serializeFormToJson($("#my-form"));
console.log("Form data:", formData);

resultsTable

Utility for displaying tabular data with pagination, sorting, and filtering capabilities.

var table = new ResultsTable($("#table-container"), {
    data: tableData,
    columns: columnDefinitions,
    pageSize: 25
});
table.render();

Event Handling

mpage.on()

Event listener for mPage-specific events. Handles custom events and system notifications.

mpage.on("patientChanged", function(patientData) {
    console.log("Patient changed:", patientData);
    // Update UI with new patient data
});

mpage.trigger()

Triggers custom mPage events. Used for communication between components.

mpage.trigger("dataUpdated", {
    type: "vitals",
    data: vitalSignsData
});

CONMANAPPNOTIFIER

Handles application notifications and alerts within the mPage context.

CONMANAPPNOTIFIER.showNotification({
    type: "success",
    message: "Data saved successfully",
    duration: 3000
});
